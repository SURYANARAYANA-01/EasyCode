All notable changes to EasyCode Dev will be documented in this file.

- Fixed Images.

## [1.0.2] - 2026-06-09

### Added

- New Java, Python, and C++ templates.
- Abstraction, Encapsulation, and Polymorphism examples.
- Exception Handling templates.
- Array Sorting and Array Reverse templates.
- String handling templates.
- File Handling, Lambda Function, and List Comprehension templates (Python).
    
### Improved

- Expanded template library across all languages.
- Updated command mappings and documentation.
- Improved template consistency and organization.

## [1.0.1] - 2026-06-07

### Added

- Java reserved class name validation.
- String operation templates.
- String reverse templates.
- Additional Java template examples.

### Improved

- README documentation.
- Marketplace screenshots and image links.
- Command documentation.
- Template organization.

### Performance

- Stability improvements.
- Performance optimizations.
- Better template loading and processing.
- Validation for missing templates and command mismatches.

### Fixed

- Java code generation for reserved class names.
- Documentation and packaging improvements.

## [1.0.0] - 2026-06-06

### Added

- Initial public release of EasyCode Dev.
- C++ code generation templates.
- Java code generation templates.
- Python code generation templates.
- Input and non-input program support.
- Ctrl + Enter shortcut for instant code generation.
- JSON-based command mapping system.
- Language-specific template organization.
- Automatic Java class name replacement using `${CLASS_NAME}`.
- Beginner-friendly programming examples.
- Support for object-oriented programming concepts.
- Support for arrays, loops, functions, operators, inheritance, and polymorphism examples.
