# Changelog

All notable changes to EasyCode Dev will be documented in this file.

## [1.2.0] - 2026-07-09

### Added

* Complete beginner-friendly HTML templates.
* Complete beginner-friendly CSS templates.
* Complete beginner-friendly JavaScript templates.
* 30 interactive front-end mini projects built with HTML, CSS, and JavaScript.
* Responsive UI templates for modern web development.
* DOM manipulation examples and concepts.
* Event handling examples using JavaScript.
* Form validation templates.
* Local Storage based projects and examples.
* Fetch API examples for working with external APIs.
* Real-world API integration projects (Weather, Movies, Pokémon, Cryptocurrency, Recipe Finder, GitHub Profile, NASA, and more).
* HTML, CSS, and JavaScript template synchronization for automatic multi-file project generation.
* Automatic creation and updating of `index.html`, `style.css`, and JavaScript files.
* Beginner-friendly web development examples covering layouts, forms, navigation, animations, responsive design, and interactive UI components.
* Comment-based code generation support for HTML, CSS, and JavaScript using Ctrl + Enter.

## [1.1.0] - 2026-06-30

### Added

* Complete beginner-friendly C++ programming templates.
* Complete beginner-friendly Java programming templates.
* Complete beginner-friendly Python programming templates.
* Basic to advanced programming concepts across all three languages.
* 31 real-world console-based mini projects for C++, Java, and Python.
* Object-Oriented Programming (OOP) examples and concepts.
* Arrays, Strings, Functions, Operators, Loops, Conditional Statements, Exception Handling, File Handling, and other core programming examples.
* JSON-based command mapping for fast code generation.
* Automatic Java class name replacement using `${CLASS_NAME}`.
* Input and non-input program templates.
* Ctrl + Enter shortcut for instant code generation.

### Improved

* Enhanced mini project templates with better structure and readability.
* More user-friendly prompts, menus, and console output.
* Standardized templates across C++, Java, and Python.
* Improved CRUD-based application workflows.
* Better template organization and consistency.
* Updated command mappings and documentation.
* Improved code formatting and maintainability.
* Optimized template loading and generation performance.

### Fixed

* Fixed template inconsistencies across all languages.
* Fixed input-handling issues in console applications.
* Corrected command mappings and template references.
* Fixed Java reserved class name validation.
* Improved overall stability and reliability.
